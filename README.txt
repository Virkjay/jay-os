JAY OS VERSION 0.3

MAIN UPGRADES
- Edit, duplicate, archive, restore and delete tasks, meetings, notes, bills and gift records.
- Five-second Undo after deletion.
- Task priorities, Top 3 editing, completion and archive views.
- Daily timeline and recurring appointments.
- Bill recurrence, payment history, next due date and reminders.
- Gift ledger grouped by person with totals.
- Notes can be pinned and archived.
- Search covers all stored information.
- Activity history records important actions.
- Existing v0.2 and v0.2.1 data uses the same localStorage key.

OUTLOOK CALENDAR
Microsoft Outlook Calendar integration is planned for Version 0.4. It requires Microsoft sign-in and Microsoft Graph configuration.

GITHUB UPDATE
1. Unzip this file.
2. Upload everything inside JayOS-v0.3 to the root of your jay-os GitHub repository.
3. Commit changes.
4. Wait 1 to 3 minutes.
5. Refresh the GitHub Pages website.
6. On iPhone, fully close Jay OS and reopen it. If the old version remains, remove the Home Screen icon, refresh the website in Safari, and add it again.

BACKUP
Open More > Data > Export before a major future update.
